# 快速开始

我们的故事将从获取代码开始

```bash
git clone https://github.com/Moechain/Moechain.git
```

## 安装

如果在你的计算机中已经安装过Node.js的话，那么事情将会变得非常简单

```bash
cd Moechain
npm i
```

## 启动

```bash
npm start 
或
node index -p [端口号]
```

如果不出意外的话，在控制台上将会有一头牛在向你问好，就像这样

```bash
 _______________________________________
< Moechain is listening on port: 2333 :-) >
```
